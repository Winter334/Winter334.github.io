:::primary
++еҫ®еҲҶз®—еӯҗжі•++пјҢдёҖз§Қз”ЁдәҺжұӮи§Јй«ҳйҳ¶зәҝжҖ§йқһйҪҗж¬Ўеҫ®еҲҶж–№зЁӢзҡ„ж–№жі•пјҢзү№зӮ№жҳҜз®ҖжҙҒй«ҳж•ҲгҖҒжҳ“зҗҶи§ЈгҖҒжҳ“дёҠжүӢгҖӮ
дёҚиҝҮеҖјеҫ—дёҖжҸҗзҡ„жҳҜеҫ®еҲҶз®—еӯҗжі•еңЁиҖғиҜ•дёӯдҪҝз”Ёж—¶еҰӮжһңи®Ўз®—й”ҷиҜҜпјҢе°Ҷйқўдёҙж— жі•еҫ—еҲҶзҡ„зӘҳеўғпјҢеӣ жӯӨе»әи®®еңЁ++иҚүзЁҝзәё++дёҠдҪҝз”Ёд»Ҙз®ҖеҢ–и®Ўз®—пјҢеҚ·йқўдёҠдҪҝз”Ёе…¶д»–зҡ„ж–№жі•гҖӮ
дёӢйқўжҳҜеҮ з§Қеҫ®еҲҶз®—еӯҗжі•зҡ„дҪҝз”Ёжғ…жҷҜгҖӮ
:::

:::warning
еҜ№дәҺеҫ®еҲҶз®—еӯҗжі•дёӯеҮәзҺ°зҡ„Dе’Ң1/DжҳҜеңЁжӯӨж–№жі•дёӯе®ҡд№үзҡ„дёҖз§Қиҝҗз®—з¬ҰпјҢеӣ жӯӨпјҢеңЁи§Јйўҳж—¶дҪҝз”Ёе®ғд»¬ж—¶йңҖиҰҒзү№еҲ«иҜҙжҳҺпјҒ
:::

е…¶дёӯDд»ЈиЎЁеҜ№еҸідҫ§еҮҪж•°иҝӣиЎҢжұӮеҜј
дҫӢпјҡ
$$
D6=0\
Dx^2 = 2x \
D\sin{x} = \cos{x} \
D^2x^2 = D2x = 2 \
(D+1) \sin{x} = D \sin{x} + \sin{x} = \cos{x} + \sin{x}
$$

$\frac{1}{D}$д»ЈиЎЁеҜ№еҸідҫ§еҮҪж•°иҝӣиЎҢз§ҜеҲҶ
дҫӢ:
$$
\frac{1}{D} 2x = x^2 \
\frac{1}{2D} \sin{x} = \frac{1}{2} \frac{1}{D} \sin{x} = -\frac{1}{2} \cos{x}\
(\frac{1}{D})^2 6x = \frac{1}{D} 3x^2 = x^3
$$
иҖҢеҖјеҫ—жіЁж„Ҹзҡ„жҳҜеһӢеҰӮ$\frac{1}{D-1}$иҝҷж ·зҡ„жҳҜж— жі•зӣҙжҺҘи®Ўз®—зҡ„

еһӢдёҖ$f(x)=e^kx$еһӢ

:::info
и§ҒkжҚўD
:::
дҫӢ: жұӮ $yвҖҷвҖҷ -3yвҖҷ + 2y = e^2x$зҡ„и§Ј
з¬¬дёҖжӯҘпјҡеҶҷеҮәеҫ®еҲҶж–№зЁӢзҡ„зү№еҫҒж–№зЁӢ$\lambda^2 - 3\lambda + 2 = 0$, з„¶еҗҺж №жҚ®зү№еҫҒж–№зЁӢеҶҷеҮә$f(D)$
иҖҢеңЁиҝҷйҒ“йўҳдёӯпјҢ$f(D)=\frac{1}{D^2 - 2D +2}$пјҢз„¶еҗҺе°Ҷ$\lambda$жӣҝжҚўжҲҗDпјҢжңҖеҗҺеҶҚжұӮеҖ’
з¬¬дәҢжӯҘпјҡи®ҫзү№и§Ј$y^=f(D)f(x)=\frac{1}{D^2 - 2D +2}e^2x$пјҢиҝҷйҮҢеҫ®еҲҶж–№зЁӢзҡ„еҸіиҫ№жҳҜд»Җд№Ҳ$f(x)$е°ұжҳҜд»Җд№Ҳ
з¬¬дёүжӯҘпјҡе°Ҷ$e^kx$дёӯзҡ„kд»Је…ҘDи®Ўз®—еҚіеҸҜеҫ—еҲ°еҫ®еҲҶж–№зЁӢзҡ„зү№и§Ј
$$
y^=\frac{1}{D^2 - 2D +2}e^2x=\frac{1}{4-4+2}e^2x=\frac 12e^2x
$$

дҫӢ1пјҡжұӮ$yвҖҷвҖҷ-2yвҖҷ+y=3e^x$зҡ„и§Ј
з¬¬дёҖжӯҘеҶҷеҮәзү№еҫҒж–№зЁӢ$\lambda^2-2\lambda+1=0$пјҢз„¶еҗҺеҶҷеҮә$f(D)=\frac{1}{D^2-2D+1}$
з¬¬дәҢжӯҘи®ҫзү№и§Ј$y^=\frac{1}{D^2-2D+1}3e^x$
з¬¬дёүжӯҘе°Ҷkд»Је…ҘDжұӮи§ЈпјҢ$y^=\frac{1}{1-2+1}3e^x=\frac103e^x$
жӯӨж—¶еҸ‘зҺ°еҲҶжҜҚдёә0дәҶжҖҺд№ҲеҠһпјҢйқўеҜ№иҝҷз§Қжғ…еҶө
йҰ–е…ҲпјҢеӣһеҲ°з¬¬дәҢжӯҘпјҢз„¶еҗҺеңЁзӯүејҸеҸіиҫ№д№ҳдёҠдёҖдёӘxз„¶еҗҺеҜ№еҲҶжҜҚиҝӣиЎҢжұӮеҜјпјҢеҫ—еҲ°$y^=x\frac{1}{2D-2}3e^x$
еҰӮжһңжӯӨж—¶еҲҶжҜҚд»Қз„¶жҳҜ0пјҢйҮҚеӨҚеҲҡжүҚзҡ„жӯҘйӘӨзӣҙеҲ°еҲҶжҜҚдёҚдёә0
иҝҷж—¶еҖҷжҲ‘д»¬еҫ—еҲ°$y^=x^2\frac123e^x=\frac32x^2e^x$

еһӢдәҢ$f(x)=sin/cosax$еһӢ

:::info
и§ҒaжҚў$D^2$,е№¶дё”еҸӘиғҪжҚў$D^2$пјҢеҸӘжңүдёҖдёӘDжҳҜдёҚиғҪжҚўзҡ„
:::
дҫӢ: $yвҖҷвҖҷ+4y=cos2x$пјҢжұӮy*
еүҚйқўжӯҘйӘӨдёҖж ·пјҢеҶҷзү№еҫҒж–№зЁӢпјҢи®ҫзү№и§Ј$y*=\frag{1}{D^2+4}\cos{2x}$
